파이썬은 변수를 선언할 때 type을 따로 선언하지 않아도 된다. <br><br>
아래의 예제를 보자.

## 예제

> ### C++
>
> ```C++
> int num = 1;
> double num = 1.23;
> char* ch = "this is string";
> ```
>
> ### Python
>
> ```python
> num = 1 # 정수
> num = 1.23 # 실수
> str = "this is string" # 문자열
> ```
>
> `C++`: 값에 알맞은 자료형을 앞부분에 선언해줘야 한다. <br>
> `Python`: 변수 선언 뒤에 원하는 값을 넣으면 파이썬이 자동적으로 자료형을 지정해준다.

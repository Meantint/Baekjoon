[BOJ_9252 - LCS 2] 풀이 str1[i] == str2[j] dp[i][j] = dp[i - 1][j - 1] + 1 str1[i] != str2[j] dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]) 위의 조건식을 문자열이 저장되어 있는 dp[i][j].second도 그대로 따른다. 이중 for문 안의 else에 등호를 >=가 아닌 >로 바꾸면 dp의 값이 달라져서 답이 달라질 수있다. 하지만 문제에 이렇게 명시되어 있기 때문에 신경쓰지 않아도 될 것 같다.

BOJ 1041 - 주사위

12 Jul 2020, 02:12
BOJ / Baekjoon / BOJ_1041

[BOJ_1041 - 주사위]  처음에 주사위의 값들이 이동할 수 있다고 생각하고 6개의 값을 sort한 후에 풀었다..(당연히 그렇게 푸는거 아님) 풀이 5개의 면 중 3개의 면이 드러나는 주사위는 4개, 2개의 면이 드러나는 주사위는 8n - 12개, 1개의 면이 드러나는 주사위는 5n^2 - 16n + 12개이다. 면이 3개인 주사위의 최소값 : 마주보는 면 중에서 더 작은 값들 3개의 합이 최소값이다. 면이 2개인 주사위의 최소값 : 하나하나 다 해봄.

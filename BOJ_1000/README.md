## 풀이

- 두 개의 숫자를 받아 덧셈의 결괏값을 출력하면 된다.

<hr>

## Code
```cpp
#include <iostream>

using namespace std;

int main()
{
    int a, b;

    cin >> a >> b;
    cout << a + b << '\n';

    return 0;
}
```

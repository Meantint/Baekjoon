Git - 명령어

  • git config –global user.name “Name”
    • 사용자 이름 정보를 입력하는 명령어
    • --global은 현재 컴퓨터에 있는 모든 저장소에서 같은 사용자 정보를 사용하도록 설정해준다.
  • git config –global user.email “Email”
    • 사용자 이메일 정보를 입력하는 명령어
    • --global은 현재 컴퓨터에 있는 모든 저장소에서 같은 사용자 정보를 사용하도록 설정해준다.
  • git init
    • 현재의 디렉터리에 저장소(repository)를 만들어준다.

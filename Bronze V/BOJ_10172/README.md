## 풀이

- 역슬래쉬 `\`를 사용해야하는 경우를 신경쓰면 된다.

## Code

```cpp
#include <iostream>

using namespace std;

int main()
{
    cout << "|\\_/|" << '\n'
         << "|q p|   /}" << '\n'
         << "( 0 )\"\"\"\\" << '\n'
         << "|\"^\"`    |" << '\n'
         << "||_/=\\\\__|" << '\n';

    return 0;
}
```

## 풀이

- 역슬래쉬 `\`를 사용해야 하는 경우만 조심하면 된다.

## Code

```cpp
#include <iostream>

using namespace std;

int main()
{
    cout << "\\    /\\" << '\n'
         << " )  ( \')" << '\n'
         << "(  /  )" << '\n'
         << " \\(__)|" << '\n';

    return 0;
}
```

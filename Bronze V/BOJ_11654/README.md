# BOJ_11654 - 아스키 코드

&nbsp;

## 문제/코드 링크

- [BOJ_11654 - 아스키 코드](https://www.acmicpc.net/problem/11654)

- [My Baekjoon Repo](https://github.com/Meantint/Baekjoon)

## 풀이

- `char` 타입 변수를 받아 `int` 타입으로 출력하면 된다.

## Code

```cpp
#include <iostream>

using namespace std;

char answer;

int main()
{
    cin >> answer;
    cout << (int)answer << '\n';

    return 0;
}
```

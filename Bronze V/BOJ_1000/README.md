# BOJ_1000 - A+B

## 문제/코드 링크

- [BOJ_1000 - A+B](https://www.acmicpc.net/problem/1000)

- [My Baekjoon Repo](https://github.com/Meantint/Baekjoon)

## 풀이

- 각 언어에 맞는 입출력을 사용하면 되는 문제.

- C++의 표준 입출력은 `cin`과 `cout`이다.

## Code

```cpp
#include <iostream>

using namespace std;

int a, b;

int main()
{
    cin >> a >> b;

    cout << a + b << '\n';

    return 0;
}
```
